Configure required SmartVideo SDK Parameters

Designate the name of the PureCloud agent's queue. Agents should be set to "On Queue" based on this input.
Navigate to PureCloud's Admin section: Admin > Account Settings > Organization Settings > Organization Details (tab) > Advanced > Organization ID to locate your Organization ID.
Find your Deployment Key by navigating in PureCloud to: Admin > Contact Center > Widgets > Deployment Key (column).
Retrieve your Tenant ID by selecting: Apps > SmartVideo_Settings > Tenant ID.
Provide the full URL address for the VideoEngager Server.
Designate the appropriate PureCloud region for your setup.

Optionally, save the current advanced settings to your Local Storage for future use. Using the 'Clear' button will erase settings from both Local Storage and the input fields.
